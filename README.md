go-marvel
=========

Go client library for Marvel API

Usage:

```
import (
  "fmt"
  
  "github.com/ImJasonH/go-marvel"
)

func main() {
  c := marvel.NewClient("my-public-key", "my-private-key")
  r, err := c.Series(2258, marvel.CommonRequest{})
  if err != nil {
    panic(err)
  }
  fmt.Println("%+v\n", r)
}
```
